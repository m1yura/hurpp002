# ПП.02 — Производственная практика (Системное администрирование)
## Тема: Пищевая промышленность (Автоматизация производственного учета)

---

## Описание

Данный репозиторий содержит материалы производственной практики ПП.02 по направлению «Системное администрирование».

В ходе практики выполнялись задачи по администрированию информационной системы предприятия пищевой промышленности, автоматизации работы системного администратора и обеспечению стабильной и безопасной работы компьютерной инфраструктуры производственного цеха и складского комплекса.

Объект автоматизации: ООО «Пищевые Технологии» (производство молочной продукции).

Практика выполнялась в среде Windows Server с использованием PowerShell и средств администрирования операционной системы.

## Содержание репозитория

### docs/
Документация по практике:
- Отчёт по производственной практике

### scripts/
Скрипты администрирования PowerShell:

| Скрипт | Назначение |
|--------|-----------|
| backup_production_db.ps1 | Резервное копирование БД производственного учета |
| restore_production_db.ps1 | Восстановление БД из резервной копии |
| create_tech_user.ps1 | Создание учетной записи для технолога |
| disable_employee.ps1 | Блокировка уволенного сотрудника |
| monitoring_production.ps1 | Мониторинг серверов производства и склада |
| disk_space_check.ps1 | Проверка свободного места на серверах БД |
| check_terminal_connect.ps1 | Проверка доступности ТСД (терминалов сбора данных) |
| windows_update_prod.ps1 | Установка обновлений на производственные серверы |
| cleanup_temp_files.ps1 | Очистка временных файлов на рабочих местах |
| antivirus_scan_prod.ps1 | Проверка антивирусной защиты серверов |
| event_logs_analysis.ps1 | Анализ системных журналов на предмет ошибок |
| print_labels_sync.ps1 | Синхронизация принтеров этикеток «Честный ЗНАК» |
| fifo_control.ps1 | Контроль остатков сырья по срокам годности |
| production_report.ps1 | Формирование отчета по выпуску за смену |

## Цель работы

Целью производственной практики являлось получение практических навыков системного администрирования в условиях пищевого производственного предприятия, автоматизация задач обслуживания ИТ-инфраструктуры, обеспечение бесперебойной работы системы производственного учета, защита данных и повышение надежности работы информационной инфраструктуры цеха и склада.

---

## Используемые технологии

- Windows Server 2019/2022 (основная платформа)
- PowerShell 5.1+ (автоматизация задач)
- PostgreSQL / MS SQL Server (базы данных производственного учета)
- UML (PlantUML / Draw.io) — моделирование процессов
- GitHub — управление версиями скриптов и документации
- 1С:Предприятие (интеграция, сопровождение)

## Результаты

В ходе практики:

- изучены задачи системного администратора на предприятии пищевой промышленности;
- разработаны скрипты автоматизации для резервного копирования, мониторинга и обслуживания;
